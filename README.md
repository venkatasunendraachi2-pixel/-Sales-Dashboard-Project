# Sales Dashboard Project (Excel)

## 📊 Overview
This project is an Excel-based Sales Dashboard created to analyze and visualize sales data.

The dashboard provides insights into total sales, orders, top-performing regions, and products using interactive charts and KPIs.

---

## 📁 Dataset
- File: `data/sales_data.csv`
- Contains 500 rows of sales data including region, product, and sales amount.

---

## 📊 Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Sales by Region
- Sales by Product
- Sales Trend Over Time

---

## 🛠 Tools Used
- Microsoft Excel
  - Pivot Tables
  - Charts
  - Slicers
  - Data Cleaning
